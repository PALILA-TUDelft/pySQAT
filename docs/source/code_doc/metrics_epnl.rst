EPNL Metric
===========

This module provides the Effective Perceived Noise Level (EPNL)
implementation used for aircraft-noise certification according to
FAR Part 36 and ICAO Annex 16.

----------
Overview
----------

* **EPNL FAR Part36** (``EPNL_FAR_Part36``)
  Derives the Effective Perceived Noise Level of an aircraft pass-by,
  including tone-correction, duration-correction, and intermediate
  metrics such as PN, PNL, and PNLT.
