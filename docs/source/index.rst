SQAT4PY Documentation
=====================

Welcome to the SQAT4PY documentation!
This documentation provides an overview of the SQAT4PY library, which is designed for audio signal processing and analysis, particularly in the context of psychoacoustics and environmental noise studies.

This library is a translation of the SQAT library from MATLAB to Python, maintaining the core functionalities while adapting to Python's ecosystem.

It is broken down into the following modules for ease of use.
